Sajt je posvecen ucenu HTML-a..


HTML, HyperText Markup Language, je standardizovani jezik koji se koristi pri struktuiranju tekstova, medija i ugradjenih objekata u web stranice i elektronsku postu.
Kao modifikovanu i pojednostavljenu verziju SGML jezika, HTML standardizuje i odrzava World Wide Web Consortium (W3C).
Originalnu verziju HTML-a kreirao je Tim Berners-Lee, a prva zvanicna verzija je izasla juna 1993 godine.

Neki od korisnih sajtova za ucenje HTML-a su:

  • W3School
  • CodeAcademy
